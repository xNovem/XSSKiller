# XSS Killer

## About The Tool :

tool to find xss vulnerability easily and quickly.

It gives the details of the vulnerability it finds as a result.

## Installation :

* `git clone https://github.com/xNovem/XSSkiller`
* `cd XSSkiller`

#### > Run : `python2 xsskiller.py`
